# 360Turns
Code for analyzing 360 turning in place 



-Folder data have the h5 from APDM
-Folder lib have the python functions to read and analize 
-Folder figures is used to save figures


TO RUN

1. Open apdm360Turn.py
2. In line 35 and 38 you can select the file to be analized
3. If you want to see the k mean detection plots, change plot_a and plot_b to value = 2 (lines 29-30)
