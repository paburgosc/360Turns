# 360Turns
Code for analyzing 360 turning in place 



-Folder data has the h5 from APDM /n
-Folder lib has the Python functions to read and analyze/n 
-Folder figures is used to save figures /n


TO RUN

1. Open apdm360Turn.py
2. In lines 35 and 3,8 you can select the file to be analyzed
3. If you want to see the k-means detection plots, change plot_a and plot_b to value = 2 (lines 29-30)
